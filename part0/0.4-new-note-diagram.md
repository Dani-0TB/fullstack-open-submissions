# 0.4: New note diagram

```mermaid

sequenceDiagram
  actor user
  participant browser
  participant server

  user--)browser: Submit form by clicking "save"
  browser->>server: POST https://studies.cs.helsinki.fi/exampleapp/new_note
  note over browser: form data sent by browser > note:"content+of+note"
  activate server
  server-->>browser: 302 found: redirect to ./exampleapp/notes
  deactivate server

  browser->>server: GET https://studies.cs.helsinki.fi/exampleapp/notes
  activate server
  server-->>browser: returns HTML document
  deactivate server

  browser->>server: GET https://studies.cs.helsinki.fi/exampleapp/main.css
  activate server
  server-->>browser: returns CSS file
  deactivate server

  browser->>server: GET https://studies.cs.helsinki.fi/exampleapp/main.js
  activate server
  server-->>browser: returns JavaScript file
  deactivate server
  note over browser: browser executes script to fetch the data from server

  browser->>server: GET https://studies.cs.helsinki.fi/exampleapp/data.json
  activate server
  note over server: server executes code to append data to server notes array and redirects to note page
  server-->>browser: return JSON data
  note over browser,server: [{ "content":"example","date":"2023-09-13T12:53:06.571Z" }, ...]
  deactivate server

  note right of browser: browser executes callback function to render list
  browser--)user: Renders site with data

```
